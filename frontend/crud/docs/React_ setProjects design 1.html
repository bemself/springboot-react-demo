14th May 2020 at 5:43pm

  • define loadProjects function inside hook

  • need to use useEffects to set projects

let [ projects, setProjects] = useState([])

Basic flow:

loadJson(read from json file)
loadProject (read from json file for single project)
create hook
    - `let [ projects, setProjects] = useState([])`
    - loadProjects (loop projects), return an array of projects data
    - `useEffect`
        - call `loadProjects`, and apply only when projectList differs (where projectList is the argument of loadProjects)
    - `createCard`: loop projects
    - render with `createCard` called
