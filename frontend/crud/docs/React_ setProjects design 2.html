React: setProjects design 方案2

14th May 2020 at 5:39pm

  • define loadProjects function outside hook

  • no need to use useEffects to set projects

Basic flow:

loadJson(read from json file)
loadProject (read from json file for single project)
loadProjects (loop projects), return an array of projects data
create hook
    - `let [ projects, setProjects] = useState(loadProjects())`
    - `createCard`: loop projects
    - render with `createCard` called

Issues:

  • createCard render不出来,见下面的comment

  • createCard 执行了两次...
